# TJ Jammalamadaka — Portfolio

Personal portfolio site for **Tarang (TJ) Jammalamadaka**, showcasing projects, experience, skills, and contact information. Built as a single-page experience with an interactive 3D scene (Three.js) and slide-out panels for content.

**Live site:** [tarang-tj.github.io](https://tarang-tj.github.io)

## Contents

- **Projects** — Selected work with links to live demos and repositories (including analytics dashboards, SQL case studies, and full-stack apps).
- **Experience** — Career timeline and role highlights.
- **Skills** — Technical stack and tooling.
- **Contact** — Email, LinkedIn, and GitHub.

## Tech

- Static **HTML / CSS / JavaScript** (no build step).
- **Three.js** (via CDN) for the background scene.
- Hosted on **GitHub Pages** from the `main` branch.

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server, for example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
